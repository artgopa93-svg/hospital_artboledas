Configuración inicial
Esta aplicación usa Firebase (gratuito) para sincronizar datos en tiempo real entre recepción y consultorio. Solo necesitas hacerlo una vez.
Paso 1 — Crear proyecto Firebase (2 minutos) 1. Ve a console.firebase.google.com
2. Clic en Agregar proyecto → ponle nombre (ej. hospital-recepcion) → Continuar hasta crear.
3. En el menú izquierdo: Compilación → Realtime Database → Crear base de datos
4. Elige la región más cercana → selecciona Modo de prueba → Habilitar.
5. Copia la URL que aparece (ej. https://hospital-xxx-default-rtdb.firebaseio.com)
Paso 2 — Pegar la URL aquí Pega solo la URL de tu base de datos. No necesitas claves API ni configuración extra.
